# A-Stellaris-Mod-of-Arknights
明日方舟牢博士为爱发电创作的一个群星模组，缓慢更新中

这是一个关于起源的mod，添加了四个和明日方舟相关的起源，分别对应泰拉人类冲破星荚来到太空，假想中乌萨斯、维多利亚和大炎统一泰拉来到太空的故事。
后面会逐步更新更多的起源、著名英杰、物种、飞升天赋等等。
mod已经发布在了创意工坊中，名字就是Arknighs_Origins。如果有什么问题两边都可以留言，我当个事儿办。
还有一个涉及到外观的mod，Arknights_Appreance，因为包含许多图片和音乐，压缩包太大发不上来，就只能去创意工坊下载了.

第一次在github上发布代码，虽然不知道有什么用，但看不少人都这么做，就跟风也发了一份。压缩包是代码的本地文件，如果要在这里下载到本地使用可能还需要一个.mod文件，但那个实际上需要的是你自己的下载路径，因此我就不上传到压缩包里去了。
